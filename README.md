Personalized Recommendation with Contextual Bandits (NLP + RL)

Course: 6INTELSY Final Project
Authors: Justin Errol L. Priniel & Jaycen John C. Carreon

Overview

This project is an intelligent news recommendation system that dynamically ranks items by capturing complex semantic features from text and adapting to shifting user preferences. It combines NLP embeddings, a 1D Text-CNN for localized feature extraction, and a Contextual Bandit (LinUCB/ε-greedy) reinforcement learning agent.

Dataset & Ethics Compliance

This project utilizes the MIcrosoft News Dataset (MIND).

License: Microsoft Research License (Non-commercial, academic use).

Privacy & Consent: The dataset consists of fully anonymized behavior logs. Microsoft ensures all data collection complies with their privacy terms, and all PII is stripped/hashed. No external data linkage will be performed.
